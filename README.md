# Week 6 - String Tasks Solution

## Tasks

1. Count Vowels - Counts the number of vowels in a given string
2. Reverse String - Returns the reversed version of a string
3. Check Palindrome - Checks whether a string is a palindrome
4. Word Frequency - Returns a dictionary with word occurrence counts
5. Longest Word - Finds the longest word in a sentence
6. Find Occurrences - Finds all starting indexes of a substring
